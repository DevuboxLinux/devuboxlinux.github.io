Introducing Devubox-MX

A midweight, simple, and stable systemd-free desktop OS.

Devubox-MX is an open-source MX Linux respin based on Devuan. It uses both MX Linux and MX AHS repositories to provide access to the latest/backports of programs and drivers, similar to how PPAs function in Ubuntu. It features a customized Openbox window manager that provides a complete desktop environment inspired by CrunchBang++, BunsenLabs, Mabox, and Archcraft.

Disclaimer: Since this distribution is in an early alpha phase, we do not recommend installing it on your main hardware. For stability, please test it in a virtual machine (VM) unless you are a tech-savvy user comfortable with pre-release software.

Estimated alpha release: Nov 20, 2025. This date is subject to change.

Midweight & Fast

Built with a customized Openbox window manager, Devubox-MX is designed to be fast and efficient. It runs smoothly on older 64-bit hardware (like the Core 2 Duo) and uses only ~600MB of RAM, similar to XFCE.

Community Driven

As an open-source project, our development is guided by user feedback. Join our forums to help shape the future of Devubox-MX.

Secure & Reliable

Based on the solid foundation of Devuan and MX Linux, Devubox-MX benefits from robust security features and a stable, reliable core system.

Preloaded with Apps for Offline Use

It's ready out of the box with multimedia codecs, games, LibreOffice, printer support, and top apps like GIMP, Inkscape, and OBS Studio.

Broad Driver Compatibility

Includes many proprietary firmware packages from the Debian `non-free-firmware` repository for out-of-the-box hardware support. With Advanced Hardware Support (AHS), it also supports a wide range of modern hardware.

Easy NVIDIA Driver Installation

A simple one-click installer is included for proprietary NVIDIA drivers (requires an internet connection).

More about Devubox-MX

The Devubox-MX combines the Openbox experience with XFCE components to create a modern look. Ready to serve both power users and beginners alike, it is midweight, fast, and resource-efficient, yet 100% powerful.

Built with user experience in mind, we've selected components to provide a midweight system that just works. Here are some of the key features of Devubox-MX:

  • Comes with lots of pre-installed applications, plus many more available through the MX Package Installer and MX Repositories using the Synaptic Package Manager.
  • Includes Rescue Tools and MX Tools from MX Linux to help repair unbootable systems and make maintenance easier.
  • The Tint2 panel has been customized with enhanced functionality, featuring an application menu, search tool, multitasking view (Exposé), software center, and buttons for logout and show desktop.
  • Added custom fonts from Devubox-MX for desktop customization.
  • The LibreOffice configuration is tweaked for a look and feel similar to Microsoft Office.
  • Easy window tiling using keyboard shortcuts.

It supports a wide range of hardware, from older 64-bit laptops to modern desktops.

Devubox-MX is Built On:

  • Base System: Devuan 6.0 "Excalibur" (Based on Debian 13 "Trixie" without systemd)
  • Repositories: MX Linux 25 Repository
  • Desktop Components: Openbox 3.6.1, Tint2 17.0.1, XFCE 4.20.1, and Picom 12.5
  • Build Tools: Built using the MX Snapshot and antiX live system tools
